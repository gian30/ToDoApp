# my-todo-project

## Project setup

```
npm install
```

Create .env file in the projects root
Fill .env file with your Supabase credentials:

```
VUE_APP_SUPABASE_URL=PUT_YOUR_SUPABASE_URL_HERE
VUE_APP_SUPABASE_ANON_KEY=PUT_YOUR_SUPABASE_ANON_KEY_HERE
```
